This short note gives a geometric perspective on conditional expectation, and a nice application of this fact to regression.

1. Conditional Expectation as Projection

Theorem 1. Let (Ω,ℱ,ℙ)(\Omega, \mathcal{F}, \mathbb{P}) be a probability space, let ℱ̃⊂ℱ\tilde{\mathcal{F}}\subset \mathcal{F} be a sub σ\sigma-algebra of ℱ\mathcal{F}, and let XX be an L2L^2 random variable. Then 𝔼(X∣F̃)\mathbb{E}(X\mid \tilde{F}) is the projection of X∈L2(ℱ)X\in L^2(\mathcal{F}) onto L2(F˜)L^2(\widetilde{F}).

Before we prove this, let’s set the measure theory aside for a second and think intuitively: a common way to introduce conditional expectation without measure theory is to call it “the best guess you would make for XX given some information.” With this in mind, it makes sense that the conditional expectation would be the point in the information you have that is “closest” to XX. But the nice thing is that this isn’t just a cute analogy it’s a precise fact.

Proof. The norm in L2(Ω)L^2(\Omega) is the square integral, so it’s sufficient to show that $$\mathbb{E}(X\mid \tilde{F})=\argmin_{Y} \mathbb{E}(X-Y)^2.$$ To do this, we’ll need a lemma.

Lemma 2. If Z∈L2(F˜)Z\in L^2(\widetilde{F}), then 𝔼(Z(X−𝔼(X∣ℱ̃)))=0.\mathbb{E}(Z(X-\mathbb{E}(X\mid \tilde{\mathcal{F}})))=0.

Proof. By Cauchy-Schwarz, ZX∈L1(Ω)ZX\in L^1(\Omega), so we have Z𝔼(X∣F̃)=𝔼(ZX∣F̃)⟹𝔼(Z𝔼(X∣F̃))=𝔼(𝔼(ZX∣ℱ̃))=𝔼(ZX),Z\mathbb{E}(X\mid\tilde{F})=\mathbb{E}(ZX\mid \tilde{F})\implies \mathbb{E}(Z\mathbb{E}(X\mid \tilde{F}))=\mathbb{E}(\mathbb{E}(ZX\mid\tilde{\mathcal{F}}))=\mathbb{E}(ZX), and then subtracting gives the result.

With this lemma in hand, take some Y∈L2(ℱ̃)Y\in L^2(\tilde{\mathcal{F}}). Then, we have 𝔼((X−Y)2)=𝔼((X−𝔼(X∣ℱ̃)+𝔼(X∣ℱ̃)−Y)2),\mathbb{E}((X-Y)^2)=\mathbb{E}((X-\mathbb{E}(X\mid\tilde{\mathcal{F}})+\mathbb{E}(X\mid\tilde{\mathcal{F}})-Y)^2), and expanding this gives 𝔼((X−𝔼(X∣F̃))2)+2𝔼([X−𝔼(X∣ℱ̃)][𝔼(X∣ℱ̃)−Y])+𝔼((𝔼(X∣ℱ̃)−Y)2).\mathbb{E}((X-\mathbb{E}(X\mid \tilde{F}))^2)+ 2\mathbb{E}([X-\mathbb{E}(X\mid \tilde{\mathcal{F}})][\mathbb{E}(X\mid\tilde{\mathcal{F}})-Y])+\mathbb{E}((\mathbb{E}(X\mid\tilde{\mathcal{F}})-Y)^2). The cross-term vanishes because of our lemma. The first term does not depend on YY. The second is clearly always positive, and is zero when Y=𝔼(X∣ℱ̃)Y=\mathbb{E}(X\mid \tilde{\mathcal{F}}), so this must be the minimum. The result follows.

2. Regression

This new paradigm has at least one nice application: understanding linear regression better.

Suppose you have some data points that are generated from jointly distributed random variables XX and YY, which satisfy Y=αX+βY=\alpha X+\beta, where α,β\alpha, \beta are constants. We observe a bunch of data from this distribution, (𝐱1,y1),⋯,(𝐱n,yn)(\mathbf{x}_1, y_1),\cdots, (\mathbf{x}_n, y_n), where yi=a𝐱i+b+𝔼psilony_i=a\mathbf{x}_i+b+\mathbb{E}psilon and 𝔼psilon∼N(0,1)\mathbb{E}psilon\sim N(0,1) is Gaussian noise.

We call 𝔼(Y∣X)=αX+β\mathbb{E}(Y\mid X)=\alpha X+\beta the true regression function, both because it is the true function and because a direct computation gives that it is the function that minimized the expected squared error of a future prediction. In practice, we don’t know the distributions of X,YX, Y, so we have to approximate the true regression function with an empirical regression function, 𝔼(Y∣X)ˆ=aX+b\widehat{\mathbb{E}(Y\mid X)}=aX+b.

To approximate the true regression function with the data, we want to approximate the projection of YY onto the σ\sigma-algebra generated by XX. But we have no idea what this σ\sigma-algebra looks like, except that it contains the data points we saw, so we just consider the space consisting of those data points instead. Then, the approximate projection will be found by choosing a,ba, b to minimize the L2L^2 distance between our empirical regression function and the approximate yy-values (we use these because we don’t know the exact yy-values). This means choosing a,ba,b to minimize ∑i=1n(a𝐱i+b−yi)2.\sum_{i=1}^n (a\mathbf{x}_i+b-y_i)^2.

But this is exactly the same as linear least squares regression. The fact that taking this statistical perspective about regression gives the same result as just deciding to minimize the sum of the squares (which can be derived in other ways as well) is not a coincidence, but rather a consequence of the fact that conditional expectations are also projections.
