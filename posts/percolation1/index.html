This is the first in a series of posts about percolation theory, which I’m studying this summer. The main text I’m using is this one, and I plan to follow it pretty closely. There should be around six posts, culminating in the proof of the Harris-Kesten theorem, a major result about percolation on ℤ2\mathbb{Z}^2. This material corresponds to the first three chapters of Bollobas, with some omissions and rearrangement.

1. The Big Picture

The idea is to consider some graph (prototypically ℤd\mathbb{Z}^d but theoretically any graph), and choose a random subgraph of this graph by selecting each edge to be part of the subgraph with probability pp. Alternatively, do the same thing with vertices instead of edges, and take the subgraph to be the one induced by the chosen vertices.

Some terminology:

  • The edges are bonds
  • The vertices are sites
  • The chosen edges/vertices are open
  • The chosen graph is the open subgraph
  • The connected components of the open subgraph are open clusters

According to this language, the two models described above are bond percolation and site percolation.

The physical motivation for this model is a liquid percolating through a solid lattice, where the open bonds/sites represent the bonds/sites that the liquid can pass through. The main question suggested by this interpretation is whether the liquid will be able to pass through the lattice completely, which corresponds to the existence of an infinite open cluster.

The Harris-Kesten theorem mentioned above answers this question for bond percolation on ℤ2\mathbb{Z}^2. It shows that for p>12p>\frac{1}{2}, there is an infinite open cluster with probability 1, and for p<12p<\frac{1}{2} there is an infinite open cluster with probability 0.

This post takes the first step toward proving this result by defining two critical probabilities that are closely related to this question, and then computing them for some simple graphs. Before we can even take that step though, we must construct a probability space (Σ,ℙ)(\Sigma, \mathbb{P}) in which to work. For a graph G=(V,E)G=(V,E), we take:

  1. Ω={0,1}|E|\Omega=\{0,1\}^{|E|} to be the set of all arrangements of open edges, where 0 represents a closed edge and 1 represents an open edge;
  2. Σ\Sigma to be the σ\sigma-algebra on Ω\Omega generated by the cylinders C(F,σ)={ω∈Ω∣ωf=σf for f∈F},F⊂E and σ={0,1}|F|;C(F, \sigma)=\{\omega \in \Omega\mid \omega_f = \sigma_f\text{ for } f\in F\},\quad F\subset E\text{ and }\sigma =\{0,1\}^{|F|};
  3. ℙ\mathbb{P} to be the probability measure on Σ\Sigma induced by ℙ(C(F,σ))=(∏f∈Fσf=1p)(∏f∈Fσf=01−p).\mathbb{P}(C(F, \sigma))=\left(\prod_{\substack{f\in F\\ \sigma_f=1}} p\right)\left(\prod_{\substack{f\in F\\ \sigma_f=0}} 1-p\right).

In everything that follows, all probabilities are defined according to this measure.

2. Critical Probabilities

For a site xx, let CxC_x be the open cluster containing xx, and let θx(p)\theta_x(p) be the probability that CxC_x is infinite.

Proposition 1. There is a critical probability pHp_H such that θx(p)=0\theta_x(p)=0 if p<pHp<p_H and θx(p)>0\theta_x(p)>0 if p<pHp<p_H, for all xx.

(The notation pHp_H is in honor of Hammersley, who first proposed this model.)

Proof. Take any two sites x,yx, y such that |x−y|=d|x-y|=d (this is the distance induced by GG). Then θx(p)≥pdθy(p)\theta_x(p)\geq p^d \theta_y(p), since if there is an infinite open cluster containing yy and a path from yy to xx, there will be an infinite open cluster containing xx as well. Thus, if θx(p)=0\theta_x(p)=0 for one site, then θx(p)=0\theta_x(p)=0 for all sites, and if θx(p)>0\theta_x(p)>0 for one site, then θx(p)>0\theta_x(p)>0 for all sites. If we show that θx(p)\theta_x(p) is an increasing function of pp, the existence of pHp_H follows.

To see the increasing behavior (which is intuitively clear), consider assigning each edge ee an independent random variable Xe∼Uni[0,1]X_e\sim \text{Uni}[0,1], and then taking it to be open if Xe<pX_e<p. If p1<p2p_1<p_2, the subgraph obtained with probability p1p_1 will always be contained in the subgraph obtained with probability p2p_2, so θx(p)\theta_x(p) must be increasing.◼\blacksquare

We now connect this pHp_H to our original question about the existence of an infinite open cluster.

Corollary 2. Let EE be the event that there is an infinite open cluster. Then ℙ(E)=1\mathbb{P}(E)=1 if p>pHp>p_H and ℙ(E)=0\mathbb{P}(E)=0 if p<pHp<p_H.

Proof. The event EE is a tail event, so by the Kolmogorov 0-1 law, it can only take the values 0 and 1. If p<pHp<p_H, then ℙ(E)≤∑sites xθx(p)=0,\mathbb{P}(E)\leq \sum_{\text{sites }x} \theta_x(p)=0, so ℙ(E)=0\mathbb{P}(E)=0. If p>pHp>p_H, then ℙ(E)≥θx(p)>0\mathbb{P}(E)\geq \theta_x(p)>0 for at least one site, so ℙ(E)=1\mathbb{P}(E)=1.◼\blacksquare

So to understand when there is an infinite open cluster, it suffices to understand pHp_H.

When p<pHp<p_H, any CxC_x is almost surely finite, but it’s expected size might not be. Set $\chi_x(p)=\E(|C_x|)$. By the same arguments as before, χx(p)\chi_x(p) increases with pp, and it is either finite for all sites or for no sites. Thus we define pT=sup{p∣χx(p)<∞},p_T=\sup\{p\mid \chi_x(p)<\infty\}, with the notation in honor of Temperley this time.

We have been deliberately ambiguous until now about whether we are working with bond percolation or site percolation in this section, and in fact we mean both. That is, there are actually four critical probabilities: pTb,pTs,pHb,pHsp_T^{\text{b}}, p_T^{\text{s}}, p_H^{\text{b}}, p_H^{\text{s}}. And since each of these depends on the choice of graph, we actually need to write pTs(G)p_T^{\text{s}}(G). This notation is a bit cumbersome though, so we refrain from using it except where absolutely necessary.

One obvious relationship between pTp_T and pHp_H is that pT≤pHp_T\leq p_H. Some more difficult relations are known, but we’ll look at those in a later post. For now, let’s do an example.

3. Computing the Critical Probabilities

Determining pTp_T and pHp_H in general is quite hard (after all, the Harris-Kesten theorem basically boils down to doing this for ℤ2\mathbb{Z}^2, and is already highly non-trivial), but there are some simple graphs for which it can be done. We do an example with bond percolation.

Suppose G=TkG=T_k is the tree in which every vertex has degree k+1k+1, except for the root which has degree kk. Call the root v0v_0 and let Tk,nT_{k,n} be the subgraph obtained by truncating this graph nn levels down from the root. We write πn\pi_n for the probability that, in bond percolation with probability pp, there is an open path of length nn from v0v_0 to a leaf of Tk,nT_{k,n}.

For there to be an open path of length nn from v0v_0 to a leaf of Tk,nT_{k,n}, there must be an open edge between v0v_0 and some other vertex, and then an open path from that vertex to a leaf. This gives πn=1−(1−pπn−1)k.\pi_n=1-(1-p\pi_{n-1})^k.

Consider the function f(x)=1−(1−px)kf(x)=1-(1-px)^k. The πn\pi_n result from iterating this function starting with π0=1\pi_0=1, so we need to study it’s fixed points. We have f(0)=0f(0)=0 and f(1)<1f(1)<1. If f′(0)>1f'(0)>1, then f(ϵ)>ϵf(\epsilon)>\epsilon for some small ϵ>0\epsilon>0, but f(1)<1f(1)<1, so there is some fixed point x0x_0. Otherwise, the unique fixed point is at x=0x=0. Concretely, f′(0)=pkf'(0)=pk, so this condition reduces to p>1kp>\frac{1}{k}.

In the case where p>1kp>\frac{1}{k}, we see that πn−1≥x0⟹πn≥x0\pi_{n-1}\geq x_0\implies \pi_n\geq x_0, and since π0=1≥x0\pi_0=1\geq x_0, we always have πn≥x0\pi_n\geq x_0. But this πn\pi_n is a lower bound on θv0(p)\theta_{v_0}(p), since if there is an infinite open cluster containing v0v_0 there must be a path to the bottom, so θv0(p)>0\theta_{v_0}(p)>0, and so pH≤1kp_H\leq \frac{1}{k}.

On the other hand, if p≤1kp\leq \frac{1}{k}, the πn\pi_n converge to the unique fixed point at 0, meaning that θv0(p)=0\theta_{v_0}(p)=0. It follows that pH=1kp_H=\frac{1}{k} for TkT_k.

We can also do pTp_T: a site at depth ℓ\ell is part of Cv0C_{v_0} with probability pℓp^{\ell}, so χv0(p)=∑y∈Tkℙ(y∈Cv0)=∑ℓ=0∞kℓpℓ,\chi_{v_0}(p)=\sum_{y\in T_k} \mathbb{P}(y\in C_{v_0})=\sum_{\ell=0}^{\infty} k^{\ell}p^{\ell}, which is finite if and only if pk<1pk<1. Thus pT=1kp_T=\frac{1}{k} as well.

A few remarks about this calculation: the fact that pT=pHp_T=p_H here is not unusual. This is true for a very general class of graphs.

Also, it turns out that this tree TkT_k has, in some sense, the lowest critical probability possible. To see this, take GG to be a graph with maximum degree Δ\Delta. If y∈Cxy\in C_x, then there is an open path from yy to xx, so we can bound χx(p)\chi_x(p) by the number of paths in GG that start at xx. Using our information about the maximum degree, this gives χx(p)≤1+∑ℓ=1∞Δ(Δ−1)ℓ−1pℓ,\chi_x(p)\leq 1+\sum_{\ell=1}^{\infty} \Delta(\Delta-1)^{\ell-1} p^{\ell}, which converges exactly when p<1Δ−1p<\frac{1}{\Delta-1}. This means that pT≥1Δ−1p_T\geq \frac{1}{\Delta-1}, and since pH≥pTp_H\geq p_T, this bound holds for pHp_H as well. In TkT_k, Δ=k+1\Delta=k+1, and this bound is attained.

So the tree TkT_k is special in more ways than one. It turns out that the more “tree-like” a graph is, the more easily pTp_T and pHp_H can be computed. Unfortunately, ℤ2\mathbb{Z}^2 is nothing like a tree, so a different approach is required to tackle it.
