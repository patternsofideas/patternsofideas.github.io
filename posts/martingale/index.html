Last quarter, I did a DRP on martingales, and at the start of this quarter, gave a short talk on what I learned. Here are the notes from that talk.

1. Conditional Expectation

Definition 1. Suppose (Ω,ℱo,ℙ)(\Omega, \mathcal{F}_o, \mathbb{P}) is a probability space, ℱ⊂ℱo\mathcal{F}\subset \mathcal{F}_o is a sub σ\sigma-algebra, and XX is a random variable.

  1. The conditional expectation of XX given ℱ\mathcal{F}, E(X∣ℱ)E(X\mid \mathcal{F}), is any ℱ\mathcal{F}-measurable random variable YY such that ∫AXdℙ=∫AYdℙ for all A∈ℱ.\int_A X\, d\mathbb{P}=\int_A Y\, d\mathbb{P}\text{ for all } A\in \mathcal{F}.

  2. If YY is another random variable, then E(X∣Y)E(X\mid Y) is defined as E(X∣ℱ)E(X\mid \mathcal{F}) where ℱ\mathcal{F} is the σ\sigma-algebra generated by YY.

Fact. The conditional expectation exists, is unique, and is integrable.

Intuitively, we can think of E(X∣ℱ)E(X\mid \mathcal{F}) as the best guess of the value of XX given the information available in ℱ\mathcal{F}.

Example 1. Some conditional expectations:

  1. If XX is ℱ\mathcal{F}-measurable, then E(X∣ℱ)=XE(X\mid \mathcal{F})=X.

  2. If XX and YY are independent, then E(X∣Y)=E(X)E(X\mid Y)=E(X).

  3. Let X1,⋯X_1,\cdots be random variables with mean μ\mu, and let Sn=X1+⋯+XnS_n=X_1+\cdots+X_n be the partial sums. Then, if m>nm>n, E(Sm∣Sn)=E(Xm+⋯+Xn+1+Sn∣Sn)=E(Xm∣Sn)+⋯+E(Xn+1∣Sn)+E(Sn∣Sn)=μ(m−n)+Sn.E(S_m\mid S_n)=E(X_m+\cdots+X_{n+1}+S_n\mid S_n)=E(X_m\mid S_n)+\cdots+E(X_{n+1}\mid S_n)+E(S_n\mid S_n)=\mu(m-n)+S_n. The idea is that you know your position at SnS_n, and you take m−nm-n steps whose sizes are, on average, μ\mu, so your best guess for your position is Sn+μ(m−n)S_n+\mu(m-n).

Now we can define martingales.

2. Martingales

A martingale is a model of a fair game.

Definition 2. Consider a filtration (increasing sequence of σ\sigma-algebras) ℱn\mathcal{F}_n and a sequence of random variables XnX_n, each measurable with respect to ℱn\mathcal{F}_n and integrable. Then, if E(Xn+1∣ℱn)=XnE(X_{n+1}\mid \mathcal{F}_n)=X_n for all nn, we say SnS_n is a martingale.

Example 2. Let X1,⋯X_1,\cdots be random variables that take only the values −1-1 and 11 with probability 1/21/2 each. Then Sn=X1+⋯+XnS_n=X_1+\cdots+ X_n is a martingale, because E(Sn+1∣Sn)=E(Xn+1+Sn∣Sn)=E(Xn+1∣Sn)+E(Sn∣Sn)=E(Xn+1)+Sn=Sn.E(S_{n+1}\mid S_n)=E(X_{n+1}+S_n\mid S_n)=E(X_{n+1}\mid S_n)+E(S_n\mid S_n)=E(X_{n+1})+S_n=S_n.

Theorem 3. If XnX_n is a martingale with supE(Xn+)<∞\sup E(X_n^+)<\infty, then XnX_n converges a.s. to a limit XX with E(|X|)<∞E(|X|)<\infty.

I’ll only sketch this proof, because even though the idea is nice, the details are a little annoying. The idea is to set up a way of betting on a martingale, show that you can’t make money off such a betting system, and then use this to draw conclusions about the martingales behavior.

Definition 4. Let ℱn\mathcal{F}_n be a filtration. A sequence HmH_m of random variables is said to be predictable if HmH_m is ℱm−1\mathcal{F}_{m-1} measurable for all mm.

That is, the value of HmH_m can always be predicted with certainty at time mm. Then, if we “bet” an amount HmH_m on the martingale XmX_m, our total winnings will be (H⋅X)n=∑m=1nHm(Xm−Xm−1).(H\cdot X)_n=\sum_{m=1}^n H_m(X_m-X_{m-1}).

If XnX_n is a supermartingale and HnH_n is bounded, then (H⋅X)n(H\cdot X)_n is a supermartingale as well.

Now, fix an interval (a,b)(a,b) and choose HnH_n in the following way: if Xn<aX_n<a, then bet 1, and continue to do so until Xn>bX_n>b. Then bet zero until you fall back below aa, and repeat. Every time you go from below aa to above bb, you will make a profit of at least (b−a)(b-a). Let UnU_n be the number of these upcrossings that occur. Then, you can use the previous fact, together with the bound on supE(Xn+)\sup E(X_n^+), to show that the number of these upcrossings is finite with probability 1. Since this is true for arbitrary (a,b)(a,b), the XnX_n must converge.

Finally, we give a brief application of the martingale convergence theorem.

Proposition 5. Let XnX_n be a sequence of random variables such that P(Xn=1)=P(Xn=−1)=12.P(X_n=1)=P(X_n=-1)=\frac{1}{2}. Then ∑j=1∞Xjj{\sum_{j=1}^{\infty} \frac{X_j}{j}} converges with probability 1.

Proof. Let Sn=∑j=1nXjj.{S_n=\sum_{j=1}^n\frac{X_j}{j}}. It’s easy to check that SnS_n is a martingale, and that E(Sn)E(S_n) is bounded (in fact, it’s always 0), so the SnS_n converge almost surely. Thus the random harmonic series converges almost surely.
