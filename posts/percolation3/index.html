The next chapter of Bollobas is a collection of probabilistic tools used throughout the book; this post presents two of these results that will be key to our proof of the Harris-Kesten theorem. Both of these results are set in the framework of weighted hypercubes, which we now introduce.

Take a set SS and its power set 𝒫(S)\mathcal{P}(S). This power set can be identified with the set of functions f:S→{0,1}f: S\to \{0,1\} by identifying A∈𝒫(S)A\in \mathcal{P}(S) with its characteristic function. In the case where SS is finite (and this is the case we care about), we can think of SS as {1,⋯,n}\{1,\cdots, n\} and 𝒫(S)=𝒫(n)\mathcal{P}(S)=\mathcal{P}(n) as the cube Qn={0,1}n.Q^n=\{0,1\}^n. This cube is the set of binary sequences of length nn, where a subset A⊂SA\subset S is associated to a sequence that has a 1 in position ii if i∈Ai\in A and a 0 otherwise.

More geometrically, we can think of QnQ^n as a graph whose vertices are 𝒫(n)\mathcal{P}(n). Two sets A,BA, B are connected in this graph if |AΔB|=1|A\Delta B|=1 (i.e. AA and BB differ at one entry in their sequences). Put yet another way, if 𝐚,𝐛∈{0,1}n\mathbf{a}, \mathbf{b}\in \{0,1\}^n are two vertices, they are connected if a−b=±𝐞ia-b=\pm \mathbf{e}_i, where 𝐞i\mathbf{e}_i is a standard basis vector. We can make QnQ^n a directed graph by saying the edge 𝐚𝐛\mathbf{a}\mathbf{b} is oriented from 𝐚\mathbf{a} to 𝐛\mathbf{b} if 𝐛−𝐚=𝐞i\mathbf{b}-\mathbf{a}=\mathbf{e}_i. We write this as 𝐚≤𝐛\mathbf{a}\leq \mathbf{b}.

Now, shifting gears, the points of QnQ^n are associated with the outcome of a sequence of nn coin tosses. If 𝐩=(p1,⋯,pn)\mathbf{p}=(p_1,\cdots, p_n) is the sequence of probabilities of success for these nn coin tosses, then $ $ induces a natural probability measure on QnQ^n.

For example, if 𝐩=(0.5,⋯,0.5)\mathbf{p}=(0.5,\cdots, 0.5) we obtain the uniform measure, and if 𝐩=(p,⋯,p)\mathbf{p}=(p,\cdots, p), we obtain ℙ(A)=p|A|(1−p)n−|A|\mathbb{P} (A)=p^{|A|}(1-p)^{n-|A|}. We write QpnQ_p^n for the cube with this measure, and it is equivalent to the standard percolation models.

We write Q𝐩nQ_{\mathbf{p}}^n for the probability space (ℙ𝐩,Qn)(\mathbb{P} _{\mathbf{p}}, Q^n) and distinguish two kinds of events. A subset U⊂QnU\subset Q^n is an up-set if a,b∈Qn,a≤b⟹b∈Ua,b\in Q^n, a\leq b\implies b\in U. Similarly, UU is a down-set if a,b∈Qn,a≥b⟹b∈Ua,b \in Q^n, a\geq b\implies b\in U. (Keep in mind that UU is a subset of QnQ^n, which is identified with 𝒫(S)\mathcal{P}(S), so a,ba, b are subsets of the original set SS.) The corresponding events are referred to as increasing and decreasing respectively.

With that set up out of the way, we can state our first lemma.

1. Harris’s Lemma

This lemma is the intuitive fact that increasing events are positively correlated. Let’s do a bit more set up.

For a set A⊂QnA\subset Q^n and t=0,1t=0,1 we can define At={(ai)i=1n−1∣(a1,⋯,an−1,t)∈A}⊂Qn−1.A_t=\{(a_i)_{i=1}^{n-1}\mid (a_1,\cdots, a_{n-1}, t)\in A\}\subset Q^{n-1}. If AA is an up-set, A0⊂A1A_0\subset A_1, and if AA is a down-set, A1⊂A0A_1\subset A_0 (because A1,A0A_1, A_0 always differ at only one position, so their difference is always a basis vector, so it’s just a matter of sign).

If we write $ $ for the probability measure on both Q(p1,⋯,pn)nQ_{(p_1,\cdots, p_n)}^n and Q(p1,⋯,pn−1)n−1Q_{(p_1,\cdots, p_{n-1})}^{n-1}, we have ℙ(A)=(1−pn)ℙ(A0)+pnℙ(A1).\mathbb{P} (A)=(1-p_n)\mathbb{P} (A_0)+p_n\mathbb{P} (A_1).

Finally, the result:

Lemma 1. Let A,BA, B be subsets of Q𝐩nQ_{\mathbf{p}}^n. If they are both up-sets or both down-sets, then ℙ(A∩B)≥ℙ(A)ℙ(B)\mathbb{P} (A\cap B)\geq \mathbb{P} (A)\mathbb{P} (B). If AA is an up-set and BB is a down-set, then ℙ(A∩B)≤ℙ(A)ℙ(B)\mathbb{P} (A\cap B)\leq \mathbb{P} (A)\mathbb{P} (B).

(This implies the correlation statement, since the first condition rearranges to ℙ(A∣B)≥ℙ(A)\mathbb{P} (A\mid B)\geq \mathbb{P} (A) and similarly for the second.)

Proof. We induct on nn. The case n=1n=1 is trivial, so suppose the result holds for n−1n-1.

If AA and BB are both up-sets or both down-sets, then either A0⊂A1A_0\subset A_1 and B0⊂B1B_0\subset B_1 or A1⊂A0A_1\subset A_0 and B1⊂B0B_1\subset B_0. This means that (ℙ(A0)−ℙ(A1))(ℙ(B0)−ℙ(B1))≥0.(\mathbb{P} (A_0)-\mathbb{P} (A_1))(\mathbb{P} (B_0)-\mathbb{P} (B_1))\geq 0. Our argument will rely on the equivalent fact that ℙ(A0)ℙ(A1)+ℙ(B0)ℙ(B1)≥ℙ(A1)ℙ(B0)+ℙ(A0)ℙ(B1).\mathbb{P} (A_0)\mathbb{P} (A_1)+\mathbb{P} (B_0)\mathbb{P} (B_1)\geq \mathbb{P} (A_1)\mathbb{P} (B_0)+\mathbb{P} (A_0)\mathbb{P} (B_1).

Now we write ℙ(A∩B)=(1−pn)ℙ(A0∩B0)+pnℙ(A1∩B1)\mathbb{P} (A\cap B)=(1-p_n)\mathbb{P} (A_0\cap B_0)+p_n \mathbb{P} (A_1\cap B_1) which is ≥(1−pn)ℙ(A0)ℙ(B0)+pnℙ(A1)ℙ(B1).\geq (1-p_n)\mathbb{P} (A_0)\mathbb{P} (B_0)+p_n\mathbb{P} (A_1)\mathbb{P} (B_1). We cleverly rewrite this as (1−pn)2ℙ(A0)ℙ(B0)+pn2ℙ(A1)ℙ(B1)+(1−p)p(ℙ(A0)ℙ(B0)+ℙ(A1)ℙ(B1)),(1-p_n)^2\mathbb{P} (A_0)\mathbb{P} (B_0)+p_n^2\mathbb{P} (A_1)\mathbb{P} (B_1)+(1-p)p(\mathbb{P} (A_0)\mathbb{P} (B_0)+\mathbb{P} (A_1)\mathbb{P} (B_1)), (yes, they’re really equal!) and use our earlier inequality to see that this is ≥(1−pn)2ℙ(A0)ℙ(B0)+pn2ℙ(A1)ℙ(B1)+(1−p)p(ℙ(A0)ℙ(B1)+ℙ(A1)ℙ(B0).\geq (1-p_n)^2\mathbb{P} (A_0)\mathbb{P} (B_0)+p_n^2\mathbb{P} (A_1)\mathbb{P} (B_1)+(1-p)p(\mathbb{P} (A_0)\mathbb{P} (B_1)+\mathbb{P} (A_1)\mathbb{P} (B_0). But this last line is equal to [(1−pn)ℙ(A0)+pnℙ(A1)][(1−pn)ℙ(B0)+pnℙ(B1)]=ℙ(A)ℙ(B),\left[(1-p_n)\mathbb{P} (A_0)+p_n\mathbb{P} (A_1)\right]\left[(1-p_n)\mathbb{P} (B_0)+p_n\mathbb{P} (B_1)\right]=\mathbb{P} (A)\mathbb{P} (B), so we conclude.

From this first piece, the other inequality follows easily: if AA is an up-set and BB is a down-set, then BcB^c is an up-set, so ℙ(A∩B)≤ℙ(A)−ℙ(A∩Bc)≤ℙ(A)−ℙ(A)ℙ(Bc),\mathbb{P} (A\cap B)\leq\mathbb{P} (A)-\mathbb{P} (A\cap B^c)\leq \mathbb{P} (A)-\mathbb{P} (A)\mathbb{P} (B^c), from the previous result. This then is ℙ(A)−ℙ(A)(1−ℙ(B))=ℙ(A)ℙ(B),\mathbb{P} (A)-\mathbb{P} (A)(1-\mathbb{P} (B))=\mathbb{P} (A)\mathbb{P} (B), as desired. ◼\blacksquare

2. The Russo-Margulis Lemma

Take an event AA in the weighted cube Q𝐩nQ_{\mathbf{p}}^n. We want to understand how ℙ(A)\mathbb{P} (A) varies with 𝐩\mathbf{p}.

For a point ω∈Qn\omega\in Q^n, we say that entry ii, ωi\omega_i, is pivotal for AA if only one of ω\omega and ω̃\tilde{\omega} are in AA, where ω̃\tilde{\omega} is ω\omega with the entry at ωi\omega_i flipped. Then, we define the influence of entry ii as βi(A)=ℙ(ωi is pivotal for A)=ℙ({ω∣ωi is pivotal for A}).\beta_i(A)=\mathbb{P} (\omega_i\text{ is pivotal for } A)=\mathbb{P} (\{\omega\mid \omega_i\text{ is pivotal for }A\}).

Lemma 2. Let AA be an increasing event in Q𝐩nQ_{\mathbf{p}}^n where 𝐩=(p1,⋯,pn)\mathbf{p}=(p_1,\cdots, p_n). Then ∂∂piℙ(A)=βi(A)⟹ddpℙ(A)=∑i=1nβi(A).\frac{\partial}{\partial p_i} \mathbb{P} (A)=\beta_i(A)\implies \frac{d}{dp}\mathbb{P} (A)=\sum_{i=1}^n \beta_i(A).

Proof. Assume without the loss of generality that i=ni=n. If we have 𝐱=(x1,⋯,xk)∈Qk\mathbf{x}=(x_1,\cdots, x_k)\in Q^k for k≤nk\leq n, write 𝐩𝐱\mathbf{p}^{\mathbf{x}} for the probability of xx in the measure induced by Q𝐩nQ_{\mathbf{p}}^n on QkQ^k. In particular, if k=nk=n, then 𝐩𝐱=ℙ({x})\mathbf{p}^{\mathbf{x}}=\mathbb{P} (\{x\}).

Also, for 𝐱∈Qn−1\mathbf{x}\in Q^{n-1}, let 𝐱+\mathbf{x}_+ and 𝐱−\mathbf{x}_- be the events formed by extending 𝐱\mathbf{x} with a 1 or 0 in the last slot, respectively. We have 𝐩𝐱++𝐩𝐱−=𝐩𝐱\mathbf{p}^{\mathbf{x}_+}+\mathbf{p}^{\mathbf{x}_-}=\mathbf{p}^{\mathbf{x}} by construction.

Now take an up-set AA, and let AaA_a be the set of 𝐱∈Qn−1\mathbf{x}\in Q^{n-1} such that 𝐱+\mathbf{x}_+ and 𝐱−\mathbf{x}_- are both in AA, and AbA_b be the same set but such that only 𝐱+\mathbf{x}_+ is. Then, ℙ(A)=∑𝐱∈Aa(𝐩𝐱++𝐩𝐱−)+∑𝐱∈Ab𝐩𝐱+,\mathbb{P} (A)=\sum_{\mathbf{x}\in A_a} \left(\mathbf{p}^{\mathbf{x}_+}+\mathbf{p}^{\mathbf{x}_-}\right)+\sum_{\mathbf{x}\in A_b}\mathbf{p}^{\mathbf{x}_+}, which, by our earlier observation, becomes ∑𝐱∈Aa𝐩𝐱+pn∑𝐱∈Ab𝐩𝐱.\sum_{\mathbf{x}\in A_a} \mathbf{p}^{\mathbf{x}}+p_n\sum_{\mathbf{x}\in A_b} \mathbf{p}^{\mathbf{x}}.

It’s clear now that ∂∂pnℙ(A)=∑𝐱∈Ab𝐩𝐱,\frac{\partial}{\partial p_n} \mathbb{P} (A)= \sum_{\mathbf{x}\in A_b} \mathbf{p}^{\mathbf{x}}, but since AbA_b is exactly the set of 𝐱\mathbf{x} for which nn is pivotal, this sum is βn(A)\beta_n(A). ◼\blacksquare

That proof is a notational mess, but the idea is simple: think of an element of QnQ^n as an element of Qn−1Q^{n-1}, plus a bit of information that tells you whether it is pivotal, and then use this decomposition to directly take the derivative.

Of course, this result is only useful if you actually understand things about the βi(A)\beta_i(A). I won’t prove it, but we have this bound: if ℙ(A)=t\mathbb{P} (A)=t and βi(A)≤δ\beta_i(A)\leq \delta for each ii, then ∑i=1nβi(A)≥ct(1−t)log(1/δ).\sum_{i=1}^n \beta_i(A)\geq ct(1-t)\log(1/\delta). This bound will be the second ingredient necessary in our proof of Harris-Kesten, which we will finally begin with in the next post.
