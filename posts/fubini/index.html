We skipped this in my analysis class, so I’m going to prove Fubini’s theorem in this post. I’m following the proof from Stein and Shakarchi’s Real Analysis with some restructuring. The basic result is that you can compute integrals in ℝm+n\mathbb{R}^{m+n} by first integrating in ℝm\mathbb{R}^m, then in ℝn\mathbb{R}^n, and the order in which you do these two integrals doesn’t matter.

To be precise, we can think of points in ℝm+n\mathbb{R}^{m+n} as (x,y)(x,y), where x∈ℝmx\in \mathbb{R}^m and y∈ℝny\in \mathbb{R}^n. Then, for each y∈ℝny\in\mathbb{R}^n, we can define the “slices” of a function ff and a set EE corresponding to yy as fy(x):ℝm→ℝ,fy(x)=f(x,y)andEy={x∈ℝm∣(x,y)∈E}.f^y(x): \mathbb{R}^m\to \mathbb{R}, f^y(x)=f(x,y)\quad\text{and}\quad E^y=\{x\in \mathbb{R}^m\mid (x,y)\in E\}. (Of course, we could define slices corresponding to x∈ℝmx\in \mathbb{R}^m as well, but it’s sufficient to do everything for yy-slices, since the proofs for xx-slices are completely analogous.)

Theorem 1. *Let ℱ\mathcal{F} be the set of functions ff on ℝm+n\mathbb{R}^{m+n} satisfying:

  1. The slice fyf^y is integrable on ℝm\mathbb{R}^m for almost every y∈ℝny\in \mathbb{R}^n
  2. The function ∫ℝmfy(x)dx\displaystyle{\int_{\mathbb{R}^m} f^y(x)\, dx} is integrable on ℝn\mathbb{R}^n for almost every y∈ℝny\in\mathbb{R}^n
  3. ∫ℝn(∫ℝmf(x,y)dx)dy=∫ℝm+nf(x,y)\displaystyle{\int_{\mathbb{R}^n}\left(\int_{\mathbb{R}^m} f(x,y)\, dx\right)\, dy=\int_{\mathbb{R}^{m+n}} f(x,y)}

Then L1(ℝm+n)⊂ℱL^1(\mathbb{R}^{m+n})\subset \mathcal{F} i.e. every integrable function is in ℱ\mathcal{F}.*

The idea of the proof is to show that ℱ\mathcal{F} contains the simple functions, and then approximate L1L^1 functions by simple functions. This approach requires two facts: that ℱ\mathcal{F} is closed under linear combinations, and that ℱ\mathcal{F} is closed under monotone limits. We start with these two proofs.

Lemma 1. A finite linear combination of functions in ℱ\mathcal{F} is in ℱ\mathcal{F}.

Proof. Suppose we have {fn}⊂ℱ\{f_n\} \subset \mathcal{F}. Then for each kk, fkyf_k^y is integrable on ℝn\mathbb{R}^n except for y∈Aky\in A_k, where μ(Ak)=0\mu(A_k)=0. If we let A=∪AkA=\cup A_k, then μ(A)=0\mu(A)=0, and if g=∑akfkg=\sum a_k f_k, then gyg^y will be integrable except when y∈Ay\in A. It follows that gg satisfies 1 and 2, and then the linearity of the integral gives 3, so g∈ℱg\in \mathcal{F}.

Lemma 2 A monotone limit of functions in ℱ\mathcal{F} is in ℱ\mathcal{F}.

Proof. Suppose we have {fn}⊂ℱ\{f_n\} \subset \mathcal{F}. We can make two simplifying assumptions: that the convergence is monotonically increasing, and that the fnf_n are non-negative. If either of these is not the case, then following argument can be applied to −fn-f_n and fn−f1f_n-f_1 to give the result.

The monotone convergence theorem tells us that limn→∞∫ℝm+nfn(x,y)dxdy=∫ℝm+nf(x,y)dxdy.\lim_{n\to \infty} \int_{\mathbb{R}^{m+n}} f_n(x,y)\, dx\, dy=\int_{\mathbb{R}^{m+n}} f(x,y)\, dx\, dy. We know that for every kk, fkyf_k^y is integrable except for y∈Aky\in A_k where μ(Ak)=0\mu(A_k)=0, and if we let A=∪AkA=\cup A_k, then μ(A)=0\mu(A)=0, and fkyf_k^y is integrable on ℝm\mathbb{R}^m for all kk except for y∈Ay\in A. Using the monotone convergence theorem again gives ∫ℝmfky(x)dx⏟gk(y)↗∫ℝmfy(x)dx⏟g(y),\underbrace{\int_{\mathbb{R}^m} f_k^y(x)\, dx}_{g_k(y)} \nearrow \underbrace{\int_{\mathbb{R}^{m}} f^y(x)\, dx}_{g(y)}, as k→∞k\to \infty.

A third application of monotone convergence gives that ∫ℝngk(y)dy→∫ℝng(y)dy\int_{\mathbb{R}^n} g_k(y)\, dy\to \int_{\mathbb{R}^n} g(y)\, dy as k→∞k\to \infty.

Now, because fk∈ℱf_k\in \mathcal{F}, we know that ∫ℝngk(y)dy=∫ℝm+nfk(x,y)dxdy,\int_{\mathbb{R}^n} g_k(y)\, dy=\int_{\mathbb{R}^{m+n}} f_k(x,y)\, dx\, dy, and this together with our previous work gives that ∫ℝng(y)dy=∫ℝm+nf(x,y)dxdy.\int_{\mathbb{R}^n} g(y)\, dy=\int_{\mathbb{R}^{m+n}} f(x,y)\, dx\, dy. The RHS is finite because ff is integrable, so gg is integrable as well. Thus g(y)<∞g(y)<\infty almost everywhere, and so fyf^y is integrable for almost every yy, and ∫ℝn(∫ℝmf(x,y)dx)dy=∫ℝm+nf(x,y).\int_{\mathbb{R}^n}\left(\int_{\mathbb{R}^m} f(x,y)\, dx\right)\, dy=\int_{\mathbb{R}^{m+n}} f(x,y). It follows that f∈ℱf\in \mathcal{F}.

Next, we show that ℱ\mathcal{F} contains the simple functions. The idea here is to start with the characteristic functions of GδG_{\delta} sets and null sets, and then use our previous work to get to simple functions.

Lemma 3. If ff is a simple function, then f∈ℱf\in\mathcal{F}.

Proof. We build this up in steps.

  1. If EE is a GδG_{\delta} set of finite measure, then χE∈ℱ\chi_{E}\in \mathcal{F}.

    We prove this in 5 sub-steps.

  1. If EE is a bounded open cube, then χE∈ℱ\chi_E\in\mathcal{F}.

    Let E=Q1×Q2E=Q_1\times Q_2 be such an open cube, where Q1,Q2Q_1, Q_2 are cubes in ℝm\mathbb{R}^m and ℝn\mathbb{R}^n respectively. It’s clear that χE\chi_E satisfies 1 and 2, so we need to do the computation to verify 3.

    For each yy, the function χE(x,y)\chi_E(x,y) is measurable in xx and integrable with g(y)=∫ℝmχE(x,y)dx={|Q1|if y∈Q1,0otherwise.g(y)=\int_{\mathbb{R}^m} \chi_E(x,y)\, dx=\begin{cases} |Q_1|\quad \text{if }y\in Q_1,\\ 0\quad \text{otherwise}\end{cases}. Thus g=|Q1|χQ2g=|Q_1|\chi_{Q_2} is also measurable and integrable, and ∫ℝng(y)dy=|Q1||Q2|.\int_{\mathbb{R}^n} g(y)\, dy= |Q_1||Q_2|. But of course, ∫ℝm+nχE(x,y)dxdy=|E|=|Q1||Q2|,\int_{\mathbb{R}^{m+n}} \chi_E(x,y)\, dx\, dy=|E|=|Q_1||Q_2|, so χE∈ℱ\chi_E\in \mathcal{F}.

  2. If EE is a subset of the boundary of a closed cube, then χE∈ℱ\chi_E\in\mathcal{F}.

    Let this set be EE. It’s still easy to check 1 and 2, so we verify 3. Because the boundary of the cube has measure zero in ℝd\mathbb{R}^d, we always have ∫ℝm+nχE(x,y)dxdy=0.\int_{\mathbb{R}^{m+n}} \chi_E(x,y)\, dx\, dy=0. On the other hand, for almost every yy, the slice EyE^y has measure zero in ℝm\mathbb{R}^m, and so the iterated integral is also zero. It follows that χE∈ℱ\chi_E\in \mathcal{F}.

  3. If EE is the finite union of closed cubes with disjoint interior, then χE∈ℱ\chi_E\in\mathcal{F}.

    The idea is to write χE=∑χAk\chi_E=\sum \chi_{A_k}, where each AkA_k is either the interior of an open cube, or a subset of the boundary of a closed cube. From 1.1 and 1.2, each χAk∈ℱ\chi_{A_k}\in\mathcal{F}, and now Lemma 2 gives that χE∈ℱ\chi_E\in\mathcal{F}.

  4. If EE is open and of finite measure, then χE∈ℱ\chi_E\in\mathcal{F}.

    Such an open set EE can be written as the countable union of almost disjoint (i.e. their intersection is null) closed cubes QjQ_j, so we define E=⋃j=1∞Qjandfk=∑j=1kχQj.E=\bigcup_{j=1}^{\infty} Q_j\quad\text{and}\quad f_k=\sum_{j=1}^k \chi_{Q_j}. Then, fk↗χEf_k\nearrow \chi_E, which is integrable because EE has finite measure, and Lemma 3 allows us to conclude that χE∈ℱ\chi_E \in \mathcal{F}.

  5. If EE is GδG_{\delta} and of finite measure, then χE∈ℱ\chi_E\in\mathcal{F}.

    Because EE is GδG_{\delta}, there are open sets GiG_i such that E=⋂n=1∞GnE=\bigcap_{n=1}^{\infty} G_n and because EE is measurable there is an open set G⊃EG\supset E. Then, let G̃k=G∩⋂n=1kGn,\widetilde{G}_k=G\cap \bigcap_{n=1}^{k} G_n, and we have a decreasing sequence of open sets G̃1⊃G̃2⊃⋯,andE=⋂k=1∞G̃k.\widetilde{G}_1\supset \widetilde{G}_2\supset\cdots,\quad\text{and}\quad E=\bigcap_{k=1}^{\infty}\widetilde{G}_k. Now, χG̃k↘χE,\chi_{\widetilde{G}_k}\searrow \chi_E, and by 1.4, χG̃k∈ℱ\chi_{\widetilde{G}_k}\in\mathcal{F}, so Lemma 3 shows that χE∈ℱ\chi_{E}\in \mathcal{F}.

  1. If μ(E)=0\mu(E)=0, then χE∈ℱ\chi_E\in\mathcal{F}.

    Because a null set EE is measurable, we can find a GδG_{\delta} set G⊃EG\supset E such that μ(G)=0\mu(G)=0. From Step 1, we know that χG∈ℱ\chi_G\in \mathcal{F}, so ∫ℝn(∫ℝmχG(x,y)dx)dy=∫ℝm+nχG=0⟹∫ℝmχG(x,y)=0,\int_{\mathbb{R}^n}\left(\int_{\mathbb{R}^m} \chi_G(x,y)\, dx\right)\, dy=\int_{\mathbb{R}^{m+n}} \chi_G=0\implies \int_{\mathbb{R}^m} \chi_G(x,y)=0, for almost every yy. This means that μ(Gy)=0\mu(G^y)=0 and Ey⊂GyE^y\subset G^y, so we have that μ(Ey)=0\mu(E^y)=0 for almost every yy, and so ∫ℝn(∫ℝmχE(x,y)dx)dy=0∫ℝdχE⟹χE∈ℱ.\int_{\mathbb{R}^n}\left(\int_{\mathbb{R}^m} \chi_E(x,y)\, dx\right)\, dy =0\int_{\mathbb{R}^d}\chi_E\implies \chi_E\in\mathcal{F}.

  2. If EE is measurable with finite measure, then χE∈ℱ\chi_E\in\mathcal{F}.

    Recall that we can find a GδG_{\delta} set G⊃EG\supset E such that μ(G\E)=0\mu(G\setminus E)=0, and now writing χE=χG−χG\E\chi_E=\chi_G-\chi_{G\setminus E} and Lemma 2 gives that χE∈ℱ.\chi_E\in\mathcal{F}.

Now, any simple function ff is a linear combination of characteristic functions of measurable sets with finite measure, so Lemma 2 shows that f∈ℱf\in\mathcal{F}.

With these three lemmas in hand, we can prove Fubini’s theorem.

Proof of Fubini’s Theorem. Suppose ff is an integrable function. We can write ff as the sum of a positive and negative part, so it is sufficient by Lemma 2 to consider the case where ff is non-negative. Because ff is integrable, there are simple functions fkf_k that converge monotonically to ff. By Lemma 4, each fk∈ℱf_k\in\mathcal{F}, and now Lemma 3 allows us to conclude that f∈ℱf\in\mathcal{F}.
